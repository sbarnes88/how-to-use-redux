# How To Use Redux
My initial attempt at implementing Redux from scratch.

Utilizes TailwindCSS, TailwindUI, HeadlessUI, and ReactJS.

## Requirements

Node version 14.17.3+
NPM version 6.14.13+

## Usage
```cli
npm install

npm start
```
